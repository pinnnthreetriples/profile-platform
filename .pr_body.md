## Testing notes

Covered auth schemas, auth route helpers, Supabase client factories, auth actions (including Supabase error and success paths), middleware matcher, and unauthenticated protected route redirects.

Auth actions tests verify:
- Invalid email/password validation
- Supabase auth error handling
- Successful login/register redirects
- Logout flow with signOut, revalidatePath, and redirect
